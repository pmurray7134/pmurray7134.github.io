FreeCodeCamp Full Stack Developer — JavaScript Curriculum (Fall 2025)
Section Steps / Pages Description
Code Editors 5 steps Introduction to code editors, setup tips, and keyboard shortcuts to enhance workflow efficiency when coding in JavaScript.
Variables & Strings 62 steps Explores variable declaration, string manipulation, and template literals for managing textual data in JavaScript.
Booleans & Numbers 37 steps Teaches basic data types, logical operations, and arithmetic expressions fundamental to control flow and calculations.
Functions 38 steps Introduces defining, invoking, and returning values from functions, including arrow functions and variable scope.
Objects 28 steps Demonstrates creating and using objects to store structured data and access properties and methods efficiently.
Arrays 35 steps Focuses on storing ordered lists, accessing elements, and using array methods such as map, filter, and reduce.
Loops 22 steps Explains iteration through data using for, while, and for...of loops to automate repetitive tasks.
JavaScript Fundamentals Review 30 steps Consolidates earlier topics through review exercises and mini-challenges to reinforce foundational skills.
Higher-Order Functions & Callbacks 37 steps Covers functions as first-class objects, passing functions as arguments, and asynchronous callback patterns.
DOM Manipulation & Events 87 steps Introduces selecting, modifying, and styling HTML elements dynamically using the Document Object Model and event listeners.
JavaScript & Accessibility 42 steps Focuses on building accessible web interactions and ensuring JavaScript-driven features meet usability standards.
Debugging 8 steps Shows how to use console methods, browser dev tools, and debugging strategies to identify and fix code errors efficiently.
Basic Regex 47 steps Introduces regular expressions for pattern matching and text validation within JavaScript programs.
Form Validation 103 steps Explains how to validate user input in forms using both HTML attributes and custom JavaScript functions.
Dates 5 steps Explores the Date object for handling timestamps, formatting, and calculating time-based events.
Audio & Video Events 55 steps Demonstrates how to manage multimedia playback using JavaScript event listeners and the HTMLMediaElement API.
Maps & Sets 37 steps Introduces the ES6 Map and Set objects for managing unique data collections and key-value pairs efficiently.
CRUD Operations 78 steps Explains Create, Read, Update, and Delete operations for data manipulation using JavaScript arrays and APIs.
Classes 68 steps Focuses on ES6 class syntax, constructors, inheritance, and encapsulation for writing modular, reusable code.
Functional Programming 112 steps Teaches pure functions, immutability, and function composition as key principles of modern JavaScript development.
Recursion 23 steps Explains how functions can call themselves to solve problems like factorials, tree traversal, and nested structures.
Asynchronous JavaScript 36 steps Covers promises, async/await, and fetch API for handling asynchronous operations like API calls and timeouts.
JavaScript Review 1 step Final comprehensive review covering syntax, structure, and debugging before progressing to advanced projects.
